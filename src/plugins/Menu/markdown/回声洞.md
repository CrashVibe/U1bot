# 回声洞
看看别人的投稿，也可以自己投稿~

## 服务信息
- **服务名**：回声洞
- **说明**：查看别人投稿的回声洞，也可以自己投稿
- **权限限定**：群聊/私聊

## 命令列表
- **回声洞**：查看别人投稿的~~发电~~回声洞，让自己放松放松罢~
- **投稿**：（仅限私聊）投稿自己的回声洞，让别人也能看到你的~~发电~~回声洞~
- **匿名投稿**：（仅限私聊）匿名投稿自己的回声洞，害羞，但就是想发表心里话 awa
- **查看回声洞记录**：查看自己投稿的~~发电~~回声洞记录
- **查看 <`编号`>**：查看指定编号的回声洞=.=
- **删除 <`编号`>**：删除指定编号的回声洞，但仅限删个人的回声洞，别人的删不了昂
